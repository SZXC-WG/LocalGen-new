LocalGen is developed in public by the SZXC-WG community. This page presents a synchronized snapshot of GitHub contributors whose commits span the core game, Qt desktop UI, map tools, built-in bots, simulator, documentation, and packaging.

GitHub's contributor data is commit-based: it may omit non-code work, co-authorship that was not recorded in Git metadata, issue triage, testing, translation, map authorship, and community support. Treat this list as one view of contribution, not a complete ranking.

See the [upstream contributor graph](https://github.com/SZXC-WG/LocalGen-new/graphs/contributors) for the canonical current view, or read [Contribute]({{< relref "contribute" >}}) to take part.
