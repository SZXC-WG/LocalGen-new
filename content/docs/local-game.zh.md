## 配置对局

在主菜单选择 **Local Game**，然后设置：

- **Game Speed：** 1–1000。每个半回合计划间隔为 `500 / speed` 毫秒，并受 Bot 处理耗时影响。速度 1 大约每秒一个完整回合。
- **Enable sounds：** 当前处于禁用状态；本地对局尚未接入声音播放。
- **Show analysis：** 添加实时兵力/领地图表，并可切换线性/对数坐标。
- **Game Map：** `Standard` 会生成新的随机地图；其他条目来自可执行文件 `maps/` 目录中的有效 `.lgmp`。
- **地图尺寸：** Standard 地图宽高范围为 1–100，默认 20×20。手工地图会将字段锁定为文件中的尺寸。
- **玩家：** 2–16。只有 P1 可以选择 `Human`；每个槽位都可以选择已编译 Bot，因此也可以观看全 Bot 对局。

当前界面最多支持一名人类。所有参与者都会获得独立队伍编号，因此比赛是自由混战。

## 选择与移动

单击格子即可聚焦。聚焦后可使用：

| 操作 | 效果 |
| --- | --- |
| 方向键或 `W A S D` | 移动焦点，并加入一次全兵力移动 |
| `I J K L` | 只移动焦点，不加入移动 |
| Shift + 方向键/WASD | 只移动焦点，不加入移动 |
| `Z` | 将下一次入队移动标记为移动一半兵力 |
| `Q` | 清空整个移动队列 |
| `E` | 删除最后一次排队移动，并把焦点退回其起点 |
| 空格 | 清除格子焦点 |
| `H` | 聚焦己方将军 |
| `G` | 聚焦己方将军并把镜头移到中央 |
| Escape | 打开投降确认 |

如果排队移动在执行时已经不合法，游戏会跳过它。山脉、瞭望塔与天文台不可通行。

## 镜头与聊天

| 操作 | 效果 |
| --- | --- |
| 按住左键拖动 | 平移棋盘 |
| 鼠标滚轮 | 以指针位置为中心缩放 |
| 触控板滚动 | 平移 |
| 双指捏合 | 缩放 |
| `9` / `0` | 缩小 / 放大 |
| `C` | 适配并居中整张地图 |
| Enter | 聚焦聊天输入；再次按 Enter 发送非空内容 |

只有存在人类玩家时才会显示聊天输入框。该面板还会记录系统提示、占领、投降与胜者，并显示回合/半回合标记。

## 读取悬浮信息

- **回合标签：** 末尾的点用于区分其中一个半回合阶段。
- **排行榜：** 显示玩家、兵力与领地。单击可折叠或展开玩家名称；死亡玩家会被遮罩，有击杀的玩家显示骷髅图标。
- **分析图：** 启用后可在兵力/领地与线性/对数视图之间切换。

人类被淘汰或比赛结束后，系统会显示完整棋盘并停止移动输入。

## 地图启动错误

手工地图必须有效且非空，还需要为所有参与者提供足够的出生格或零兵力空白平地。如果检查失败，请返回设置减少玩家数量，或修改地图。
