## 开始制作地图

在主菜单选择 **Create Map**。编辑器默认创建 10×10 棋盘；宽度和高度滑块支持 1–100，调整尺寸时会保留重叠区域。

普通单击绘制一个格子，按住右键拖动可连续绘制，按住左键拖动则平移棋盘。鼠标滚轮用于缩放，`C` 适配整个棋盘，方向键上/下选择上一个或下一个工具栏项目。

## 格子工具

| 工具 | 行为 |
| --- | --- |
| 山脉 | 不可通行地形 |
| 瞭望塔 | 不可通行；相邻占领军可获得 5×5 范围视野 |
| 天文台 | 不可通行；从相邻军队向外投射最多八格直线视野 |
| 沙漠 | 占领后不会获得每 25 回合一次的全局兵力加成 |
| 沼泽 | 被占领时每回合减少一兵；降到零时失去归属 |
| 出生点 | 将军起点；可留空表示灵活分配，也可标记队伍 `A`–`Z` |
| 城市 | 初始兵力范围 -9999 至 9999；编辑器默认 40 |
| 中立军 | 放置 -9999 至 9999 的中立兵力；零会转为空白格 |
| 点亮 | 切换任意格子的全局可见状态 |
| 擦除 | 恢复为空白且未点亮的格子 |

当前本地对局会给每名玩家分配独立队伍。出生标签仍会影响固定出生组与随机队伍之间的映射，但设置界面不会创建盟友队伍。

## 元数据

可折叠侧栏可以编辑：

- 标题；
- 作者；
- 创建日期与时间；
- 纯文本描述。

本地对局会在地图选择器中使用 `.lgmp` 去除首尾空格后的标题。标题为空时回退到文件名；标题重复时会附加文件名以便区分。

## 打开、导入与保存

| 操作 | 支持的输入/输出 |
| --- | --- |
| 打开 | `.lgmp`、旧版 `.lg`、官方地图 `.json` |
| 导入 | 按准确标题获取公开 generals.io 地图；需要联网 |
| 保存 | 仅 `.lgmp` 或 `.lg` |

保存 `.lg` 时会显示警告，因为该格式无法保存标题、作者、创建时间与描述。官方 JSON 只能导入；转换后请另存为 `.lgmp`。

JSON 导入器会验证尺寸、格子数量、标题与 ISO-8601 `created_at`。无法识别的官方格子代码会被标成错误，便于在保存前编辑。

## 安装完成的地图

将地图保存为 `.lgmp`，再复制到 `LocalGen-new` 旁的 `maps/` 目录。重新打开本地对局窗口即可刷新列表。请确保地图为预期玩家数量提供足够的明确出生点或零兵力空白格。

格式状态以及回放/设置限制见[关联文件]({{< relref "docs/associated-files" >}})。
