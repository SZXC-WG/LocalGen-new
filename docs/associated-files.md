# Local Generals.io v6 (Qt) 关联文件

| 扩展名 / 文件名  | 性质 | 备注                                                         |
| :--------------: | :--: | :----------------------------------------------------------- |
|      `.lg`       | 地图 | 适用于 LG v5 的地图文件。需要配合相应 ini 文件使用。         |
|     `.lgmp`      | 地图 | 适用于 LG v6 的地图文件。自带 metadata。                     |
|      `.lgr`      | 回放 | 一般回放文件，包含复刻对局过程所必需的全部数据。             |
|     `.lgra`      | 回放 | 高级回放文件。在 `.lgr` 基础上额外存储一系列信息，如每回合每个玩家的移动队列等。 |
| `settings.lgsts` | 配置 | 适用于 LG v5 的配置文件。默认应为隐藏状态。                  |
| `settings.json`  | 配置 | 适用于 LG v6 的配置文件。默认应为隐藏状态。                  |
